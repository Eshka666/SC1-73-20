Блок навигации

  • Ширина 15 ем
  • Должен присутсвовать цвет фона
  • Распологается слева
    • Блок навигации наполните контентом достаточным, чтобы он начал захватытвать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу равным 2ем, при помощи свойств margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу равным 2ем, при помощи свойств margin-bottom Картинка вставлена при помощи тега, плаввание организованно по правому краю внутри данной статьи для этой картинки

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
